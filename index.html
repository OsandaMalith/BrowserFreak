BrowserFreak is a Automated Password Dumper for Web Browsers

    Features

  • Can dump passwords of Chrome, Firefox, Opera, Safari and IE
  • Run as Administrator to function this properly
  • Can be used during pen-tests
  • Requires internet connection
  • Added some voice ;)
